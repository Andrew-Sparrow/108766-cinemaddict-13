# Project «Cinemaddict»

### Used technologies
- OOP
- MVP
- Inheritance and polymorphism
- Abstract classes
- REST API
- Promise
- webpack


### functionality:
- filtering the list of movies
- sorting movies
- add, delete movie's comments
- interaction with the server
